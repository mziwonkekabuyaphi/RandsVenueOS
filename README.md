# RandVenueOS

**RandVenueOS** is a comprehensive venue management and event dashboard system built for managing ticket sales, queues, cashless events, and more. This system integrates multiple modules such as Rands Tickets, Ticket Channels, QLess Queue Management, E-commerce, Shisha Lounge management, Rands Wallet, and Private Label branding.

---

## Features

### Dashboard
- Live ticket sales stats
- Queue monitoring (regular & VIP)
- Revenue overview and analytics
- Event performance insights

### Ticketing
- Event Tickets management
- Box Office operations
- Ticket Shop & WhatsApp/SMS ordering
- Ticket Scanner App
- Event Self-Checkin

### Queue Management (QLess Q)
- Queue tracking
- WhatsApp, SMS, QR Code joins
- Queue Kiosk & TV Display
- Staff calling app

### E-commerce
- Orders & Products management
- Order Pick-up Queue
- Staff Calling App
- Queue Display
- E-commerce Settings

### Cashless Events
- Rands Wallet
- VVIP POS Tab System
- Smart Counter Orders
- Shisha & Butcher Shop Orders

### Private Label Branding
- Customize system name, logo, favicon
- Primary & secondary colors
- Live dashboard preview

### Security
- Admin & Staff login
- Role-based access
- Demo credentials for testing
